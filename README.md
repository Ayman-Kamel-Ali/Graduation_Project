# Heart Attack Prediction Using Big Data & Machine Learning.

---

## 🎯 Project Overview

This is a comprehensive **graduation project** focused on building an intelligent **Heart Disease Prediction System** using advanced data warehousing techniques, machine learning models, and real-time data streaming. The project integrates multiple modern data technologies including Apache Spark, Kafka, MongoDB, and Power BI for analytics and visualization.

The system leverages a comprehensive heart disease dataset to train predictive models that can identify individuals at risk of heart disease, providing valuable insights for healthcare professionals and patients.

---

## 🚀 Project Goals

1. **Data Warehousing**: Design and implement a robust data warehouse (DW) for storing and managing comprehensive heart disease data
2. **Predictive Analytics**: Develop accurate machine learning models to predict heart disease risk
3. **Real-Time Processing**: Implement stream processing using Apache Kafka for real-time data ingestion and processing
4. **Data Augmentation**: Apply GAN (Generative Adversarial Networks) techniques to augment training datasets
5. **Visualization & Reporting**: Create interactive dashboards using Power BI for stakeholder insights
6. **Model Optimization**: Implement multiple algorithms (Random Forest, etc.) and compare performance
